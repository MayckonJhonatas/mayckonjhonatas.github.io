Política de Privacidade do CodeBlocks+

Aplicativo educacional de programação em blocos para Arduino e ESP32.
Última atualização: 15/11/2025
O CodeBlocks+ foi desenvolvido com foco em uso educacional, offline e local. Não coletamos, não armazenamos e não enviamos dados pessoais dos usuários para servidores externos.

Esta Política de Privacidade descreve como o aplicativo CodeBlocks+, desenvolvido por Mayckon Jhônatas, lida com as informações dos usuários. Ao utilizar o aplicativo, você concorda com os termos descritos abaixo.

1. Coleta de Informações

O CodeBlocks+ não solicita, não coleta e não armazena informações pessoais como nome completo, endereço, documentos, fotos, contatos, localização precisa ou qualquer outro dado sensível.

1.1. Dados armazenados localmente

O aplicativo armazena apenas informações necessárias para o seu funcionamento, e sempre de forma local no dispositivo, tais como:

  • Projetos criados pelo usuário (arquivos de programação em blocos, por exemplo em formato XML);
  • Configurações de preferência do aplicativo (tema, idioma, último projeto aberto etc.);
  • Histórico de conexões com placas e dispositivos (quando aplicável), apenas enquanto o app estiver em uso.

Esses dados não são enviados para a internet, não são compartilhados com terceiros e permanecem apenas no dispositivo do usuário.

1.2. Conexões com dispositivos externos

O CodeBlocks+ pode se conectar a placas e módulos (como Arduino, ESP32 e módulos Bluetooth) para envio de código e comunicação serial. Essas conexões acontecem somente quando o próprio usuário inicia a ação, por exemplo:

  • Conexão via Bluetooth com módulos como HC-05, HC-06 ou ESP32;
  • Conexão via Wi-Fi (OTA) para envio de código ao ESP32;
  • Conexão via USB (quando aplicável) para gravação de código em placas.

Todo o tráfego acontece apenas entre o aplicativo e o dispositivo conectado (placa, módulo, robô), sem uso de servidores remotos.

2. Permissões Utilizadas

O CodeBlocks+ solicita apenas as permissões necessárias para que os recursos de programação e comunicação com as placas funcionem corretamente. Essas permissões são utilizadas exclusivamente para a finalidade descrita abaixo.

  • Bluetooth / Bluetooth LE: usado para pesquisar e se conectar a dispositivos como módulos Bluetooth e placas ESP32. Sem essa permissão, não é possível enviar código ou dados via Bluetooth.
  • Localização aproximada (quando exigida pelo sistema): em algumas versões do Android, o acesso ao Bluetooth exige permissões de localização. O CodeBlocks+ não utiliza esses dados para rastreamento ou monitoramento; a permissão é apenas uma exigência técnica do sistema operacional.
  • Armazenamento / Arquivos: utilizado para salvar e carregar projetos criados pelo usuário no dispositivo. Nenhum arquivo é enviado para a nuvem ou para serviços de terceiros pelo aplicativo.

Em todos os casos, as permissões podem ser gerenciadas diretamente pelo usuário nas configurações do sistema operacional (Android).

3. Compartilhamento de Informações

O CodeBlocks+ não compartilha dados com:

  • empresas terceiras;
  • serviços de anúncios;
  • plataformas de análise de uso (analytics);
  • servidores externos ou bancos de dados na nuvem.

Toda a atividade do app é voltada ao uso local e educacional, sem coleta ou envio automático de informações.

4. Serviços de Terceiros

O aplicativo não integra, até o momento, serviços como Firebase, Google Analytics, redes de anúncios ou outros mecanismos de rastreamento. Caso futuramente algum serviço de terceiros seja integrado, esta Política de Privacidade será atualizada para informar claramente o tipo de dado utilizado.

5. Segurança

Como o CodeBlocks+ não coleta dados pessoais nem envia informações para servidores externos, os riscos relacionados à exposição de dados são significativamente reduzidos.

Ainda assim, recomenda-se que o usuário mantenha o dispositivo protegido com senha, biometria ou outro método de segurança oferecido pelo sistema operacional, e evite instalar o aplicativo por meios não oficiais.

6. Direitos do Usuário

Como não há coleta de dados pessoais pelo aplicativo, não há cadastro interno para edição, exportação ou exclusão de dados em servidores. Todas as informações são controladas diretamente pelo usuário em seu próprio dispositivo.

O usuário pode, a qualquer momento:

  • Excluir projetos e arquivos criados dentro do CodeBlocks+;
  • Limpar dados do aplicativo nas configurações do sistema;
  • Desinstalar o aplicativo, removendo completamente os dados locais relacionados a ele.

7. Alterações nesta Política de Privacidade

Esta Política de Privacidade poderá ser atualizada periodicamente para refletir melhorias do aplicativo ou alterações legais. Sempre que houver mudança relevante, a nova versão será disponibilizada nesta mesma página.
