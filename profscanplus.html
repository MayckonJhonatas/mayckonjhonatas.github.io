Política de Privacidade

1. Quem é o responsável

O responsável pelo aplicativo ProfScan+ (“Aplicativo”) é Mayckon Jhônatas.
Contato: mayckonjhonata@gmail.com

2. Quais dados o aplicativo pode coletar

Dependendo dos recursos utilizados, o Aplicativo pode coletar/usar:

  • Dados informados pelo usuário: turmas, listas de alunos, atividades, gabaritos e resultados.
  • Dados gerados no uso: respostas registradas, estatísticas e relatórios.
  • Câmera: usada para leitura/escaneamento (ex.: cartões/folhas OMR). A câmera é usada apenas durante a leitura.
  • Armazenamento do dispositivo: para salvar/exportar arquivos (ex.: imagens/PDFs/relatórios), quando disponível.
  • Dados técnicos (opcional): informações básicas de funcionamento (ex.: erros), para melhorar estabilidade.
Ajuste o texto acima conforme o ProfScan+ realmente faz. Se não coletar “dados técnicos”, remova essa parte.

3. Como usamos os dados

  • Permitir o funcionamento do Aplicativo (cadastros, correções, resultados e relatórios).
  • Salvar e organizar informações localmente no dispositivo.
  • Melhorar desempenho e corrigir falhas (quando houver coleta de logs).

4. Compartilhamento de dados

O ProfScan+ não vende dados pessoais. O compartilhamento pode ocorrer apenas:

  • Quando você exporta arquivos (ex.: enviar relatório por outro app).
  • Se você ativar recursos que dependam de serviços externos (por exemplo, backup/sincronização, se existir no futuro).

5. Armazenamento e segurança

  • Por padrão, os dados podem ser armazenados localmente no dispositivo (ex.: banco local).
  • Adotamos medidas razoáveis para proteger informações, mas nenhum sistema é 100% seguro.

6. Retenção e exclusão

  • Você pode excluir turmas/atividades/resultados dentro do Aplicativo (quando o recurso existir).
  • Ao desinstalar o Aplicativo, dados locais podem ser removidos conforme o comportamento do sistema operacional.

7. Crianças e adolescentes

O Aplicativo é voltado para uso educacional. Se houver dados de estudantes (ex.: nome em lista), recomenda-se que o uso seja feito por professores/responsáveis, seguindo normas da escola e legislação aplicável.

8. Permissões do dispositivo

  • Câmera: para leitura de cartões/OMR.
  • Armazenamento/Arquivos (quando aplicável): para salvar/exportar relatórios/arquivos.

O usuário pode gerenciar permissões nas configurações do dispositivo.

9. Alterações desta política

Podemos atualizar esta Política. A versão mais recente ficará disponível no endereço onde este documento foi publicado.

10. Contato

Em caso de dúvidas: mayckonjhonata@gmail.com

O ProfScan+ oferece ferramentas para apoio ao trabalho docente (ex.: escaneamento, correção e relatórios). Ao utilizar o Aplicativo, você concorda em usá-lo de forma lícita e adequada.

2. Responsabilidade sobre os dados

Você é responsável pelas informações inseridas (ex.: listas de alunos e gabaritos) e por seguir regras da sua instituição e legislação aplicável.

3. Limitações

4. Propriedade intelectual

O Aplicativo, marcas e materiais associados pertencem ao desenvolvedor, salvo quando indicado o contrário.

5. Alterações
